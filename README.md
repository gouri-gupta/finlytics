# 🚀 Finlytics – Financial Analytics Platform

Finlytics is a full-stack MERN application that transforms raw financial transactions into meaningful insights.  
Instead of just tracking money, it helps users understand their spending behavior and make smarter decisions.

---

## 🚀 Live Demo

👉 Try Finlytics here:  
🌐 https://finlytics-lemon.vercel.app  

📡 API Endpoint:  
⚙️ https://finlytics-backend-t4r0.onrender.com

## 🌟 Features

### 💰 Transaction Management
- Add, edit, delete transactions
- Supports both **income & expense**
- Add notes for better context

### 🔍 Advanced Filtering
- Filter by:
  - Type (Income / Expense)
  - Category
  - Date
- Pagination support for large data

### 📊 Dashboard & Insights
- Income vs Expense (Pie Chart)
- Category-wise spending (Bar Chart)
- Monthly trends (Line Chart)
- Real-time financial summary

### 🎨 UI/UX
- Clean and modern UI
- Fully responsive design
- Dark mode support
- Smooth animations & transitions

## 🛠️ Tech Stack

**Frontend:**
- React.js
- Tailwind CSS
- Recharts

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB (Aggregation Pipelines used)

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository
```bash
git clone https://github.com/your-username/finlytics.git
cd finlytics
```

### 2️⃣ Backend Setup
```bash
cd backend
npm install
npm start
```

### 3️⃣ Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

---


## 🔐 Environment Variables

Create a .env file in backend:
```bash
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```
